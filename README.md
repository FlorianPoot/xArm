# xArm
LewanSoul xArm

## Features
* API to easely program the xArm.
* Inverse kinematics with IKPy.
* Compatible with Raspberry Pi.
* xArm Launcher: to launch differents programs and get servos positions.

## Requirements
* Python 3.7
* Following Python packages: ikpy, hidapi
